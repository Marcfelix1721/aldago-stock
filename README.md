# Aldago Stock · Gestión de Almacén

Herramienta interna de gestión de almacén para Aldago serveis i instal·lacions.

## Estructura del proyecto

```
aldago-stock/
├── public/
│   ├── index.html
│   └── manifest.json
├── src/
│   ├── index.js
│   └── App.jsx
├── package.json
└── vercel.json
```

## Despliegue en Vercel

1. Sube esta carpeta a un repositorio de GitHub
2. Importa el repositorio en vercel.com
3. Vercel detecta automáticamente que es React
4. Pulsa Deploy

## Instalar en el móvil (PWA)

1. Abre la URL de Vercel en Chrome (Android) o Safari (iPhone)
2. Pulsa "Añadir a pantalla de inicio"
3. La app funciona como una app nativa
